Node-Red-Fiap-CNPJ
==================

Utilizado API de Busca CNPJ com node-red.

- Plataforma de low-code para entrada de usuário e utilização de dashboard realizado em node-red para buscar dados do CNPJ via API e retorná-los ao usuário.
